# AlphaLens AI

> Financial Document Intelligence Platform

AlphaLens AI transforms financial reports into a semantically enriched knowledge base using deterministic document parsing, AI-powered enrichment, vector embeddings, and Retrieval-Augmented Generation (RAG).

## Features

- Deterministic Markdown parsing
- Coverage validation
- Stable chunk IDs
- AI-powered semantic enrichment
- Jina Embeddings
- ChromaDB
- Financial document retrieval
